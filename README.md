## ✨ Особенности

- 🎮 **Автоматическая генерация квестов** 
- 📊 **3 уровня сложности** 
- 🎁 **Гибкая система наград** с поддержкой команд
- ⚡ **Поддержка RW+** — пропуск квестов для юзеров рв+

---

## 📋 Команды

### 👾 Игровые команды

| Команда | Описание |
|---------|----------|
| `/quest` | Открыть меню актуальных квестов |
| `/quest1` | Открыть меню квестов 1-го уровня |
| `/quest2` | Открыть меню квестов 2-го уровня |
| `/quest3` | Открыть меню квестов 3-го уровня |
| `/skipquest <номер>` | Пропустить квест *(требуется RW+)* |

### 👑 Административные команды

| Команда | Описание |
|---------|----------|
| `/quest admin help` | Показать все админ-команды |
| `/quest admin reset all_quests` | Обновить все квесты |
| `/quest admin reset all_rewards` | Удалить все награды |
| `/quest admin reward add <уровень> command <линия> <команда>` | Добавить командную награду |
| `/quest admin reward remove <уровень> <линия>` | Удалить награду |
| `/quest admin skip all_quests <1\|2\|3>` | Пропустить все квесты уровня |

---

## 🔐 Права доступа (Permissions)

| Право | Описание |
|-------|----------|
| `quests.admin` | Доступ к административным командам |
| `quests.skipquest` | Возможность пропускать квесты *(RW+ подписка)* |

---

## 🚀 Установка

1. Скачай последний релиз из [Releases](https://github.com/lightld/CandyQuests/releases)
2. Помести `CandyQuests.jar` в папку `plugins/` твоего сервера
3. Перезапусти сервер или используй `plugman load CandyQuests`
4. Настрой плагин под свой сервер
5. Готово! 🎉
